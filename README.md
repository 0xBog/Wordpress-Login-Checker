# Wordpress Login Checker

## Installation

Install Wordpress Login Checker

```bash
  git clone https://github.com/0xBog/Wordpress-Login-Checker

  cd Wordpress-Login-Checker-main

  pip install -r requirements.txt

  python main.py
```
